Current Features (Week 1 Implementation)

This application is a custom C# Windows Forms text editor inspired by Notepad, designed with a more personalized and visually engaging user experience.

Implemented features include:

A fully functional text editor using a RichTextBox

Ability to create a new document

Ability to open existing .txt files

Ability to save and save-as text files

Single-file editing scope, similar to Windows Notepad

Clean and intuitive menu system (File menu)

Status bar for document context

Travel-themed UI with background imagery

Integrated calendar that allows the user to select a date and see a countdown

Dynamic countdown messaging (future date, today, or past date)

Background music player for a more relaxing writing experience

Clear layout focused on usability and readability

All code is fully commented line-by-line in my own words to explain functionality and logic.

Project Goal

The goal of this project was to build a functional Notepad-style application while improving overall Windows UX through thoughtful layout design, personalization, and creative features that go beyond the default Notepad experience.
