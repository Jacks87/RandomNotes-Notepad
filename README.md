RandomNotes is a Windows Forms (C#) text editor inspired by Notepad, with a more personalized and travel-themed user experience.

## Features (Current)
- RichTextBox-based text editor
- File menu with New, Open, Save, Save As, and Exit
- Trip countdown using a calendar
- Background music player
- Clean, intuitive Windows Forms UI
- Fully commented source code written in my own words

## Planned Features (Future)
- Multiple note tabs
- Autosave functionality
- Theme switching (light/dark)
- Search and replace
- Word count and status bar enhancements
- Custom fonts and text colors

## Tech Stack
- C#
- Windows Forms (.NET)
- Visual Studio
